<h3>课程介绍：</h3>
<p><a title="查看与 正则表达式 相关的文章" target="_blank">正则表达式</a>，又称正规表示法、常规表示法（英语：Regular Expression，在代码中常简写为regex、regexp或RE），计算机科学的一个概念。正则表达式使用单个字符串来描述、匹配一系列符合某个句法规则的字符串. 在很多文本编辑器里，正则表达式通常被用来检索、替换那些符合某个模式的文本</p>
